1. XSL (Extensible Stylesheet Language) - это язык разметки, который используется для описания внешнего вида XML-документов. 
XSLT (Extensible Stylesheet Language Transformations) - это часть XSL, являющаяся языком преобразования XML-документов.
2. Основное назначение технологии XSLT - это преобразование XML-документов из одного формата в другой, с помощью заданных правил
 и шаблонов.
3. Элемент <xsl:template> используется для определения шаблона, который будет применяться к определенным элементам или узлам 
XML-документа.
4. Значение match="/" указывает, что данный шаблон должен применяться к корневому элементу XML-документа.
5. Чтобы подключить XSLT к XML, нужно в XML-документе указать ссылку на XSLT-файл с помощью инструкции <?xml-stylesheet?>.
 Например: <?xml-stylesheet type="text/xsl" href="stylesheet.xsl"?>
6. Элемент <xsl:stylesheet> используется для определения начала XSLT-документа. Он содержит информацию о версии XSLT, 
пространстве имен и другие настройки.
7. Элемент <xsl:apply-templates> используется для применения шаблонов к соответствующим узлам XML-документа. Он указывает 
XSLT-процессору, какие шаблоны должны быть применены.
8. Элемент <xsl:sort> используется для сортировки узлов или элементов XML-документа. Атрибуты, которые могут быть указаны 
в <xsl:sort>, включают select, order, data-type и lang.
9. Для сортировки с условиями можно использовать элемент <xsl:choose> вместе с <xsl:when> и <xsl:otherwise>. С помощью них
 можно задать различные правила сортировки в зависимости от определенных условий.
10. Элемент <xsl:otherwise> используется внутри элемента <xsl:choose> для указания действий, которые должны быть выполнены,
 если ни одно из условий <xsl:when> не выполняется.
11. Элемент <xsl:when> используется внутри элемента <xsl:choose> для указания условий, которые должны быть проверены. Если
 условие выполнено, то выполняются действия, указанные внутри <xsl:when>.
12. К XSL относятся язык XSLT, язык XSL-FO (Extensible Stylesheet Language - Formatting Objects) и язык XPath (XML Path Language),
 который используется для навигации по XML-структурам.
13. Шаблоны преобразований в XSLT строятся с использованием элементов <xsl:template> и <xsl:apply-templates>. <xsl:template> 
определяет шаблон, а <xsl:apply-templates> указывает, где и когда применять этот шаблон.
14. Алгоритм преобразования XML-документа с помощью языка XSLT включает следующие шаги: загрузка и анализ XML-документа, 
обработка и применение XSLT-шаблонов, создание выходного документа на основе примененных шаблонов.
15. Элемент <xsl:value-of> используется для вывода значения определенного узла или атрибута XML-документа в результате 
преобразования.
16. Элемент <xsl:for-each> используется для выполнения набора действий для каждого выбранного узла XML-документа. 
Внутри <xsl:for-each> могут быть использованы другие элементы XSL, такие как <xsl:value-of>, <xsl:if>, <xsl:sort> и другие.
